# Secret Message Decryption Terminal

An interactive Caesar Cipher decryption game built with **React**, **TypeScript**, and **Vite**.

This project is a web version of my original Python terminal application while preserving the original game logic and encryption algorithm.

## Live Demo

https://secret-message-decryption-game-b69akz4kp-hru3475.vercel.app/

## Features

- Interactive terminal interface
- Caesar Cipher decryption
- Original Python logic preserved
- Animated terminal UI
- Responsive design
- View original Python source code
- Restart game

## Technologies Used

- React
- TypeScript
- Vite
- CSS
- Python (Original Project)

## Run Locally

```bash
npm install
npm run dev
```
